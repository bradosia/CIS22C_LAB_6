Detailed Description

template<class T>
class List< T >

A Singly Linked List ADT which will be composed of one or more nodes. Implement the most common linked-list behaviors as explained in class - new list, add anywhere, delete anywhere, find anywhere, count of items in the list, empty the list, etc.
